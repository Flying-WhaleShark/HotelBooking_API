# Hotel Booking API

This is a Spring Boot API project for hotel booking system.

## Getting Started

### Prerequisites
- Java 17
- Maven

### Running the Application
```bash
./mvnw spring-boot:run
JAVA_HOME=/Users/ryota/.jdk/jdk-25/Contents/Home ./mvnw spring-boot:run
```

### Building the Application
```bash
./mvnw clean package
```

### mvn test
```bash
./mvnw test
```
